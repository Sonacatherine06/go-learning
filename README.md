# Go Learning

This repository contains beginner-friendly Go programs created while learning Go.

## Programs

- Hello World
- Calculator
- Factorial
- Fibonacci Series
- Prime Number Checker
- Reverse String
- Palindrome Checker
- Array Sum
- Student Struct


## Technologies Used

- Go (Golang)

## How to Run

```bash
go run hello.go
```

Replace `hello.go` with the name of any Go program you want to run.
